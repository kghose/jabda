Jabda
A desktop personal journal (diary)

Jabda is a simple desktop program with which you can keep a personal journal. Jadba's interface is through your favorite browser.

Installation (Windows)
  1. Get the application bundle
  2. Unzip it where ever you like.
  3. Double click on pylog.exe to start the diary server
Installation (Mac) Using (First start)
  1. Start your browser and enter 
    http://localhost:3010/config
  2. Enter the full path to where you would like to save your diary file. e.g. 
    /Users/myname/mydiary.sqlite3
    Entering 
    "./mydiary.sqlite3"
    will save the diary in the same folder as the program, which is useful if you put the program on a USB stick or portable HDD which you carry round with you.
